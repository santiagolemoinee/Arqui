# Arqui - Ecosystem Operating System for Real Estate

Arqui is a comprehensive digital infrastructure designed to professionalize the Real Estate industry. It unifies capital, development, and community into a single shared source of truth.

## 🚀 Tech Stack

- **Framework**: [Vite](https://vitejs.dev/) + [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Project Structure

- `src/App.jsx`: Main application containing all landing page sections and logic.
- `public/`: Assets, images, and static 3D viewers.
- `src/components/ui/`: Reusable UI components like the Pain Points Carousel.

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SWLConsultingIT/Arqui.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🏗️ Modules

- **Arqui Build**: Total Construction Control.
- **Arqui State**: Marketplace with Traceability.
- **Arqui Home**: Building Social Network.
- **Arqui Management**: Smart Building Management.
- **Arqui Capital**: Investment Infrastructure.
